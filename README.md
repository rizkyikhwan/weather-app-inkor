# Weather App

## How do we run this repository

1. Clone or Download this repository
2. Open project

### `yarn install`
3. Type "yarn install" on the terminal

### `yarn dev`
4. Type "yarn dev" for running the project

### Dependencies
NPM Version 9.8.1

1. Vite
2. ReactJS
3. Tailwind CSS
4. Sonner
5. React-icons
6. Jest Testing
